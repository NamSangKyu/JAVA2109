생성자(Constructor)
	클래스 객체가 될때 제일 먼저 호출하는 메서드
	모든 클래스가 생성자를 이용하여 객체 생성
	생성자는 단 한번만 실행
	메서드명이 클래스명과 동일함

메서드 중복 정의(Overloading)
	메서드 이름이 중복이되면 컴퓨터가 누굴 호출해야 될지 모름
	이름이 같다면 매개변수 형태를 보고서 구분
	1. 매개변수 개수가 다름
	2. 매개변수 개수 같다면 매개 변수 타입이 다르게하면 됨
	메서드 명을 중복해서 쓰기위한 기술
	 
필드(Field) --> C++ 에서는 멤버 변수
	클래스 내에서 선언된 변수
	객체의 데이터에 해당하는 부분	
	
키워드 this
	클래스 내에서 자기 자신 객체를 나타내는 별칭
	생성자나 setter/getter에서 주로 필드와 동일한 매개변수를 구분하기 위해 사용한다 
	
	
메서드
	클래스가 해야되는 행동을 표현
	미리 작성한 코드(필요할 떄마다 불러다 쓰는 개념)
	
	*리턴타입 : 변수타입 결과을 줄때 사용자에게 이 메서드의 결과값이 어떤 타입입니다. 라고 알려줌
	
	리턴타입   메서드명(매개변수){
		실행할 코드
		return 값; --> 결과값 시킨 사람에게 결과 보고
	}
